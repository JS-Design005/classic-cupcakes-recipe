# Classic Cupcakes

## Purpose
Thank you for taking the time to see my project! This recipe website is a folio piece to showcase my work. I am learning about Semantic HTML and CSS. One of the challenges of this project was using grids to keep the footer at the bottom of the page. 

## Hypothetical Brief
Taste Mag wants a new website to archive thier recipies that is similar in design to the magazines people know and love. Font styles and photography needs to match, and the recipies should be simple and easy to navigate. 

Our target audience is quite broard, ranging from elderly to young cooks. It must be beginner friendly. Including allergen advice at the bottom is an important safety feature that needs to be clear, but not too distracting.
